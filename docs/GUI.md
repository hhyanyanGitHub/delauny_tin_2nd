# Windows GUI 演示说明

可执行文件：`build/dterrain_demo.exe`。

## 操作

| 控件 | 功能 |
|---|---|
| 模拟10万 | 生成约10万点的起伏地形并建网 |
| 模拟100万 | 生成约100万点，用于大数据交互演示 |
| 清空 | 清除当前三角网 |
| 查询模式 | 单击显示最近顶点和覆盖三角形 |
| 插入模式 | 单击插入一个按模拟地形函数计算高程的新点 |
| 删除模式 | 单击删除 XY 距离最近的顶点 |
| 框选放大 | 按住鼠标左键拖出矩形，松开后将选区放大并按画布比例适屏 |
| 全图 | 恢复到数据完整范围 |
| 导入XYZ | 导入 `.xyz/.txt/.csv` 散点并立即自动构网 |
| 保存网格 | 保存 `.dtmesh/.txt` 文本三角网或 `.dtin` 二进制网格 |
| 打开网格 | 打开并校验 `.dtmesh/.txt`，或加载 `.dtin` |

鼠标滚轮以指针位置为中心缩放。右键或中键拖动平移。在“框选放大”模式下，
按住左键拖出青色虚线选择框，松开后自动放大；选择过程中按 `Esc` 可取消。

XYZ 文件每个有效行包含 `x y z`，也接受逗号、分号和制表符；空行和以 `#`
开头的行会被忽略。文本三角网格式见 [TEXT_FORMATS.md](TEXT_FORMATS.md)。
便携目录附带的 `sample_data/sample_points.xyz` 可直接用于导入测试。

## 显示颜色

- 蓝—绿—黄—棕—白：由低到高的地形线框；
- 红色：编辑前被删除或受影响的三角形；
- 黄色：影响区域边界；
- 绿色：编辑后新增的三角形和边；
- 洋红色：查询点所在三角形；
- 白色圆点：查询点的最近网格顶点。

## 性能策略

DLL 始终精确查询所有与当前视口相交的三角形。为避免 GDI 在全图绘制数十万条
线段时阻塞，GUI 最多抽样显示约45,000个三角形；放大后会自动重新查询并显示更
完整的局部细节。此限制只属于演示渲染器，不影响三角网数据和查询结果。
